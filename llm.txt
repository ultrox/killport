# killport

Kill processes on a port. Works on Linux, macOS, Windows.

Install: npm install -g @ma.vu/killport
Run without install: npx @ma.vu/killport 3000

Usage:
  killport 3000          # kill process on port
  killport 3000 --force  # SIGKILL
  killport --peek        # list all listening ports

Zero runtime dependencies. Pure Node.js.
